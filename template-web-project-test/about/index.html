About
This project is made as part of a course at City University of New York.
-
It is intended as a assignment to create a network of pages to tell a story or poem by using coding (html+css).
-
The illustrations and inspiration for this project come from the graphic novel "Rids" by Sidsel Mørch about Gustaf Munch-Petersen
-
It is created by Benjamin Mørch, a graduate student in Digital Humanities at CUNY.
-
If you wish to get in contact, write to bmoerch@gradcenter.cuny.edu
